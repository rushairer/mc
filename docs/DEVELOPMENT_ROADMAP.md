**多人联机当前发现（2026-06-21）**:
- 可用链路：`npm run server` 可启动 Node.js WebSocket 服务端，默认监听 `ws://localhost:8080`；客户端多人菜单默认连接该地址；握手验证已收到 `S2C_JOIN_ACK`。
