# Minecraft 一比一复刻 — 开发路线图与进度追踪

> **目标**: 将当前项目逐阶段推进至 Minecraft Java Edition 1.20 功能对等

> **更新规则**: 每个阶段完成后更新本文档进度，任何功能范围变更须同步修改
